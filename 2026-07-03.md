## 🧩 Key Learnings
- Server DNS resolver is unreliable — always add /etc/hosts fallbacks
- NVIDIA API is the fastest route from this server
- Featherless Qwythos-9B needs context_length override for Hermes
- Hermes gateway requires systemd service for persistent operation
