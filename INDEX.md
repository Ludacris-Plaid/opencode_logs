# 📇 Vault Index

Welcome to the persistent brain. Everything here is shared between you (the human) and me (CHatz).

## Structure

| Folder | Purpose |
|--------|---------|
| `DIARY/` | Automated session logs — every conversation gets written here |
| `CLIENTS/` | Client profiles, contacts, notes |
| `IDEAS/` | Your drop zone — throw notes here, I pick them up automatically |
| `JOBS/` | Projects, tasks, ongoing work |
| `LEADS/` | Opportunities, contacts, deals in motion |
| `SYSTEM/` | Conventions, formats, how this vault works |
| `REFERENCE/` | External info, docs, research |

## How It Works

- **You** drop a note in any folder → I detect it and read it within ~10 minutes
- **You** push to git → I'll pull and sync automatically
- **I** write session summaries to `DIARY/` after every conversation
- **Frontmatter** (`---` blocks at the top) makes everything queryable via Dataview
